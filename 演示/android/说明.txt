1. 演示的app使用 java 编写，java 调用 jni 接口， jni 的接口是用C++封装调用 libofdpdfsigner.so 动态库。
   因此，用户在调用时，调用的是 java 接口。

2. 整个运行过程，没有连接任何网络，仅仅在android虚拟机中运行。

3. 演示的功能：
   功能按钮依次为：打开ofd文件，打开pdf文件，位置盖章，盖骑缝章，关键字盖章，验证文档中的签章。
   操作流程为：
     打开ofd文件 -》位置盖章 -》盖骑缝章 -》关键字盖章 -》 验证文档中的签章
     打开pdf文件 -》位置盖章 -》盖骑缝章 -》关键字盖章 -》 验证文档中的签章

4. 具体源代码请到 android目录中查看
