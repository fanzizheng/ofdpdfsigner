# ofdpdfsigner
功能强大的 ofd/pdf 签章工具，支持格式互转，支持文档加密及完整性保护。
转换支持：
        ofd -》 pdf
        ofd -》 png
        ofd -》 jpeg
        ofd -》 svg

        pdf -》 ofd
        pdf -》 png
        pdf -》 jpeg
        pdf -》 svg
